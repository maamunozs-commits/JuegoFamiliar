# Lúdica Sur - Catálogo de Juegos de Mesa

Proyecto desarrollado para la actividad de Semana 1 de la asignatura **Desarrollo Full Stack II**.

## Datos del estudiante

- **Nombre:** Matías Muñoz
- **Carrera:** Ingeniería en Desarrollo de Software
- **Asignatura:** Desarrollo Full Stack II
- **Institución:** Duoc UC Online

## Descripción del proyecto

Lúdica Sur es una PYME ficticia dedicada a promocionar la venta de juegos de mesa.  
El proyecto consiste en un sitio web estático que muestra una página principal con categorías y páginas internas con fichas de juegos.

El sitio fue desarrollado utilizando solamente **HTML** y **CSS**, sin JavaScript, sin frameworks, sin Bootstrap, sin backend, sin login y sin carrito de compras.

## Objetivo de la actividad

Crear una interfaz gráfica clara y ordenada para presentar el catálogo de juegos de mesa de una PYME, aplicando los contenidos iniciales estudiados en la asignatura.

## Contenidos aplicados

- Estructura base de documentos HTML.
- Uso de etiquetas semánticas como `header`, `nav`, `main`, `section`, `article` y `footer`.
- Creación de hipervínculos entre páginas internas.
- Organización de archivos y carpetas dentro de un proyecto web.
- Vinculación de una hoja de estilos CSS externa.
- Uso de variables CSS con `:root`.
- Diseño de tarjetas para categorías y productos.
- Aplicación de colores, bordes, espaciados y sombras.
- Uso de imágenes locales en formato SVG.
- Adaptación responsive mediante media queries.
- Aplicación de animaciones CSS con `@keyframes`.
- Uso de Git y GitHub para versionar y compartir el proyecto.
- Organización de tareas mediante tablero Trello.

## Páginas del sitio

- `index.html`: página principal con presentación de la PYME y listado de categorías.
- `familiares.html`: catálogo de Juegos Familiares.
- `estrategia.html`: catálogo de Juegos de Estrategia.
- `cartas.html`: catálogo de Juegos de Cartas.
- `cooperativos.html`: catálogo de Juegos Cooperativos.

## Características principales

- Sitio web estático.
- Diseño limpio y fácil de revisar.

- Máximo 3 juegos por categoría.
- Imágenes locales en formato SVG.
